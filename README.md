# sage-syntax.github.io

This is the public repository containing the source code and assets for the **Sage Syntax** web platform.

### 🌐 Live Platform
Visit the live website for full details, core concepts, and services:
👉 **[https://sage-syntax.github.io](https://sage-syntax.github.io)**

---

### 🛠️ Local Development

To preview or edit the website locally:
1.  Open `index.html` in your web browser.
2.  Or run a local development server from this directory:
    ```bash
    # Using Python
    python3 -m http.server 8000
    
    # Using Node.js / Live Server
    npx live-server
    ```
